2. Likelihood function

With the likelihood function the conditional probability p$(r|\lambda,H)$ , it is a function of both $\boldsymbol{r}$ and $\boldsymbol{\lambda}$ . For fixed value of $\boldsymbol{\lambda}$, p$(r|\lambda,H)$ describes the distribution of $\boldsymbol{r}$ , while for fixed data r, p$(r|\lambda,H)$ defines the lilkelihood of $\lambda$, it doesn't have to be normalized.

An interesting property of Poission distribution is both the expectation value and the variance of r are $\boldsymbol{\lambda}$, denoted as E(r) = Var(r) = $\boldsymbol{ \lambda}$.

3.Posterior probability

By employing the Bayes' rule:

$p(\lambda|r,H)=\frac{p(\lambda|H) p(r|\lambda,H)}{p(r|H)}$¶

where the denominator $p(r|H)$, also called the evidence, is caculated by integral all the possible values of $\lambda$.

$p(r|H) = \int p(r,\lambda|H) d\lambda = \int p(\lambda|H) p(r|\lambda,H) $

Since the prior $p(\lambda|H) $ is uniform distributed, we obtain the posterior distribution of $\lambda$ ,

p$(\lambda|r,H)= p(r|\lambda) = \frac{e ^{-\lambda} \lambda ^{r}}{r!}$.

Though it has the same funtion form of the likelihood function p$(r|\lambda,H)$, unlike $\boldsymbol{r}$ , which can only be positive integers, $\boldsymbol{\lambda}$ has positive continuous values, so p$(\lambda|r,H)$ is a probability density function.

It can be interpreted as given the observations r, combining our prior knowledge about $\lambda$, we will be able to estimate the posterior probability distribution of $\lambda$ .

First of all, let's simulate the data pairs (x,r).¶

$\boldsymbol{x}$ is a random variable ( e.g.the energy of electrons in nuclear Physics ), $\boldsymbol{r}$ is a variable dependent ( e.g.the detected counts in the detector ), for each data pair $(x_b,r_b)$, (b = 1,2,3 ...B), B is the maximum number of outcomes.

$ r_b \sim$ Poisson($ \lambda_b $).

where, Poisson($ \lambda_b $) = p$(r_{b}|\lambda _{b}) = \frac{e ^{-\lambda _{b}} \lambda_{b} ^{r_{b}}} {r_{b}!} $.

With this in mind, we build up a model, say $\boldsymbol{\lambda} = \theta_{0} + \theta_{1} \boldsymbol{x} $ ,¶

$\boldsymbol{\lambda}$ = {$\lambda_1, \lambda_2, \lambda_3 \cdots \lambda_B $}, $\boldsymbol{x}$ = {$ x_1, x_2 ,x_3 \cdots x_B $}.

where $\theta_0 $ and $ \theta_1$ are parameters we are interested.

Define the random variable $\boldsymbol{x}$ has value $\boldsymbol{x}$ $\in$(0,20), $ \theta_0 =1 $ , $ \theta_1 = 2 $ . Next, we use the model $\boldsymbol{\lambda} = \theta_{0} + \theta_{1} \boldsymbol{x} $ to generate $\boldsymbol{\lambda}$ , then the corresponding $\boldsymbol{r}$ follows from $\boldsymbol{r}$ = Poisson($\boldsymbol{\lambda}$).

Now, here comes out a question, what if we don't know the value of parameters $\theta_0 $ and $ \theta_1$ ?

For example, after an experiment , we obtain a bunch of data (x,r), we want to describe the relationship between x and r. Then we need to evaluate $\boldsymbol{\lambda}$ in order to find the best fitting.

Generally, there are two ways of solving this problem. The fundamental difference between the two methods is how we treat the model parameters $\boldsymbol{\lambda}$ we are interested , in this case, $\boldsymbol{\lambda} \sim \boldsymbol{\theta}$, $\theta_0 $ and $ \theta_1$ are hyperparameters . The traditional frequentist method treat the value of $\boldsymbol{\theta}$ fixed, and derive quantities from random repeated events, the mathematical notation is the conditional distribution :$p(D|\theta,H)$. The other way is Bayesian method, our belief about parameters $\boldsymbol{\theta}$ change according to fixed observed data, mathematical speaking is $p(\theta|D,H)$. In this post I focus on this approach.

Since we have no idea about the hyperparameters $\theta_0 $ and $ \theta_1$ , the best way to do is to say they are uniform distributed. Also we know the counts are integers, then we can say likelihood function is a poisson pmf, assuming the model is $\lambda=\theta_0 + \theta_1 x$ .¶
